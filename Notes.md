## Added features
- Implemented read, create, update & delete to tasks
- Check box on the task list will update the isDone value to true/false
- Ability to add & delete tasks on the list
- Also added filtering to the tasks by users, adding new task will assign it to the selected user in the dropdown
- Added the create, read, update & delete to user API

- Data validation for fields
- Pagination in case of large amount of task
- Could have used more additional fields for filtering & sorting
